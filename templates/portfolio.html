BlogPosts - Flask

"Blog Spots" es una plataforma de blogs desarrollada con Flask y SQLAlchemy. Permite a los usuarios registrarse, iniciar sesión, crear, editar y eliminar publicaciones. La interfaz utiliza HTML, CSS y Bootstrap, gestionados por Jinja2. SQLite se emplea como base de datos en el entorno de desarrollo.

API Contactos - Flask

La aplicación de Contactos, creada con Flask y SQLAlchemy, ofrece una API REST para gestionar contactos con operaciones CRUD. Utiliza Flask como framework web y SQLAlchemy para la interacción con una base de datos SQLite. Permite la creación, lectura, actualización y eliminación de contactos a través de endpoints dedicados.

Sistema farmaceutico - Python

La aplicación "Sistema de Farmacia" utiliza PyQt5 para crear una interfaz gráfica intuitiva. Permite gestionar productos farmacéuticos mediante funciones como registro, actualización y eliminación en una base de datos SQLite. Destaca visualmente la importancia de las fechas de vencimiento con colores diferenciados. La interfaz incluye pestañas para facilitar la navegación entre las distintas funciones, ofreciendo una experiencia eficiente en la gestión del inventario.

Sistema de rotiseria - Django-Vue

La aplicación de la rotisería combina Django en el backend y Vue.js en el frontend para ofrecer una plataforma completa de gestión de productos. Permite a los usuarios ver y agregar productos de manera fácil y eficiente a través de una interfaz intuitiva y receptiva. Con Django, se proporciona una API RESTful para la comunicación entre el frontend y el backend, asegurando una experiencia de usuario coherente y una gestión eficaz del inventario. GitHub
